# Crypto
Live prices of cryptocurrencies.
Update every couple of seconds.

## Technologies Used
- Kotlin
- Jetpack Compose
- MVVM architecture
- Coroutines, Flow
- Hilt dependency injection
- Retrofit, Moshi

## Setup
Download and open in Android Studio, then install on your Android device.
